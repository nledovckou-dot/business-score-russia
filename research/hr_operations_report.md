### 3. Дефицит кадров — что сложнее всего закрыть

| Позиция | Сложность найма | Причины дефицита |
|---------|-----------------|------------------|
| **Senior Data Scientist (NLP/LLM)** | Критическая | Соотношение спроса/предложения 3.2:1. Конкуренция с BigTech (Яндекс, Сбер). Средний срок закрытия: 3-6 мес |
| **ML Engineer (MLOps/LLMOps)** | Очень высокая | Новая специализация, мало готовых специалистов. Зарплатные ожидания на 25% выше предложений |
| **Senior отраслевой аналитик** | Высокая | Узкий пул экспертов с глубоким знанием отрасли + аналитическими навыками. Часто уходят в консалтинг/inhouse |
| **Data Engineer (антибот, распределённые системы)** | Высокая | Конкуренция с e-commerce и маркетплейсами. Особенно дефицитны специалисты по обходу защит и real-time ETL |
| **DD-консультант (senior)** | Средне-высокая | Узкая экспертиза, требуется опыт в Big4 или инвестбанке. Переход в корпоративный M&A |
| **Junior аналитик** | Низкая | Большой поток выпускников, легко закрывается через стажировки |
| **Бизнес-аналитик (middle)** | Средняя | Много кандидатов, но качественных — немного. Конкуренция с IT-компаниями |

**Ключевой факт**: к 2026 году более 90% организаций столкнутся с дефицитом квалифицированных IT-кадров. В сфере ИИ дисбаланс особенно острый.

**Рекомендации по закрытию дефицитных позиций:**
- HH.ru — основной канал, но Senior DS/ML получают по 5-10 офферов одновременно
- Telegram-каналы (Data Science Jobs, ML Jobs Russia) — для техспециалистов
- Хабр Карьера — для инженеров данных
- Рекомендации и хантинг — для senior отраслевых экспертов
- Стажёрские программы + обучение — для закрытия junior позиций в перспективе 6-12 мес

### 1. Типичный процесс маркетингового исследования

#### Этапы и сроки

| Этап | Содержание | Срок |
|------|-----------|------|
| 1. Бриф и постановка задачи | Уточнение целей, гипотез, KPI, аудитории | 2-3 дня |
| 2. Дизайн исследования | Выбор методологии (кабинетное / полевое / смешанное), разработка инструментария | 3-5 дней |
| 3. Кабинетное исследование | Анализ открытых источников: ФНС, ЕГРЮЛ, отраслевые отчёты, СМИ, конкуренты | 1-2 недели |
| 4. Полевое исследование (если нужно) | Опросы, глубинные интервью, фокус-группы, mystery shopping | 2-4 недели |
| 5. Обработка и анализ данных | Статистическая обработка, моделирование, перекрёстная проверка | 1-2 недели |
| 6. Подготовка отчёта | Визуализация, выводы, рекомендации, презентация | 1 неделя |
| 7. Согласование и доработка | Презентация клиенту, Q&A, финальные правки | 3-5 дней |

**Общий срок**: 4-8 недель (кабинетное), 6-12 недель (с полевым компонентом)

### 2. Процесс Due Diligence

#### Этапы

| Этап | Содержание | Срок | Команда |
|------|-----------|------|---------|
| 1. Определение scope | Цели проверки, приоритетные области, формат, NDA | 2-3 дня | Инициатор + руководитель проекта |
| 2. Запрос документов | Чек-лист: учредительные, финансовые, договоры, суды, лицензии | 3-7 дней | Проектный менеджер |
| 3. Финансовый DD | Анализ отчётности, активов, обязательств, cash flow, нормализация EBITDA | 2-3 недели | Финансовые аналитики (2-3 чел) |
| 4. Юридический DD | Корпоративная структура, суды, интеллектуальная собственность, лицензии | 2-3 недели | Юристы (1-2 чел) |
| 5. Налоговый DD | Налоговые риски, проверки, оптимизация | 1-2 недели | Налоговые консультанты (1-2 чел) |
| 6. Коммерческий/маркетинговый DD | Рынок, конкуренты, клиентская база, unit-экономика | 2-3 недели | Аналитики (2-3 чел) |
| 7. Операционный DD | Бизнес-процессы, IT-системы, персонал, ключевые риски | 1-2 недели | Операционные консультанты |
| 8. Подготовка заключения | Синтез, Red flags, рекомендации, оценка | 1 неделя | Вся команда |

**Общий срок**: 3-6 недель (стандартный), до 3-6 месяцев (крупные холдинги), 10-14 дней (экспресс)

**Команда**: 5-12 человек (типичный DD), включая юристов, финансистов, оценщиков, отраслевых экспертов

**Стоимость**: от 80 до 300+ человеко-часов. При ставке 5 000-15 000 ₽/час — от 400 000 ₽ до 4 500 000+ ₽.

**Важно**: в России нет закона, регулирующего процедуру DD — порядок определяется целями инициатора.

### 4. Автоматизация vs ручная работа

#### Что уже автоматизировано (или легко автоматизируется)

| Процесс | Инструмент/подход | Экономия времени |
|---------|-------------------|-----------------|
| Сбор данных ФНС/ЕГРЮЛ | API + парсинг (Python, Scrapling) | 90% — минуты вместо часов |
| Мониторинг конкурентов | Автоматические парсеры, RSS, Brand Analytics | 80% |
| Анализ соцсетей (подписчики, ER) | API соцсетей, Telemetr.me, скраперы | 85% |
| SEO-аудит | SEMrush/Serpstat автоотчёты | 70% |
| Генерация отчётов (шаблонные) | Jinja2, автоформатирование, LLM-суммаризация | 60-70% |
| Скрапинг цен и ассортимента | Scrapling, Selenium, API маркетплейсов | 90% |
| NLP-анализ отзывов | LLM (GPT-5, Gemini) + sentiment analysis | 80% |
| Финансовый экспресс-анализ | Автокоэффициенты из ФНС-данных | 75% |

#### Что требует ручной работы (и не будет полностью автоматизировано)

| Процесс | Почему не автоматизируется | Требует |
|---------|---------------------------|---------|
| Экспертные интервью | Нужен человеческий контакт, адаптация вопросов | Senior аналитик |
| Валидация гипотез | Отраслевая экспертиза, контекст | Отраслевой эксперт |
| Стратегические рекомендации | Творческое мышление, учёт неявных факторов | Senior+ аналитик |
| Юридический DD | Интерпретация договоров, оценка рисков | Юрист |
| Презентация клиенту | Убеждение, Q&A, подстройка под аудиторию | Руководитель проекта |
| Mystery shopping | Физическое присутствие | Полевой агент |
| Перцептуальные карты | Экспертная оценка позиционирования | Аналитик + клиент |

#### Баланс автоматизации в 2026

**Текущее состояние отрасли:**
- 45% российских маркетологов используют ИИ для анализа данных и отчётности
- К 2026 году до 70% рекламных процессов управляются нейросетями
- Компании с AI-аналитикой показывают рост эффективности на 20-30%

**Оптимальное соотношение для аналитической компании:**
- **70% автоматизации** — сбор данных, первичная обработка, шаблонные отчёты, мониторинг
- **30% ручной экспертной работы** — стратегия, валидация, клиентская коммуникация, сложный анализ

**Вывод для BSR**: автоматизация рутины (скрапинг, ФНС, SVG-визуализация, LLM-суммаризация) — это ключевое конкурентное преимущество. При штате 3-5 человек можно выдавать объём работы, сопоставимый с агентством на 15-20 человек, за счёт AI-конвейера.

## КЛЮЧЕВЫЕ ВЫВОДЫ

### HR:
1. **Минимальный жизнеспособный штат**: 5-8 человек (CEO + 2-3 аналитика + 1 DE + 1 DS + 1 PM)
2. **ФОТ минимум**: 1.5-2.5 млн ₽/мес (Москва, gross) = 18-30 млн ₽/год
3. **Критический дефицит**: Senior DS (NLP/LLM), ML Engineer, отраслевые эксперты
4. **Стратегия**: растить junior внутри + хантить senior на проектной основе
5. **Зарплатные ожидания кандидатов на 25% выше медианных предложений** — планировать бюджет с запасом

### Операционка:
1. **Стоимость инструментов**: 726 000 – 1 652 000 ₽/год (минимальный стек)
2. **Типичный проект (маркетинговое исследование)**: 80 000 – 500 000 ₽, 4-8 недель
